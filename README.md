# 屏幕区域监控 + OCR + 自动点击

## 安装

```bash
pip install -r requirements.txt
```

首次运行无需联网，RapidOCR 模型在 pip wheel 内联（~13MB onnx 模型），安装即用。

## 运行

```bash
python main.py
```

## 使用

1. 在「监控区域」面板添加区域（输入名称 + top/left/width/height，或点「框选区域」用鼠标拖拽框选）
2. 在「点击点」面板添加点击点（输入名称 + X + Y + 顺序，或点「拾取坐标」）
3. 在顶部填写关键词（用 `|` 分隔，如 `成功|完成|已就绪`）
4. 设置周期（秒）
5. 点击「开始」

- 截图保存到 `src/pic/YYYYMMDD/HHMMSS_<区域>_<序号>.png`
- OCR 命中结果追加到 `src/csv/YYYYMMDD.csv`（UTF-8 with BOM，Excel 直接打开）
- 配置自动保存到 `config.json`，重启后自动加载

## 拾取坐标 / 框选区域

两种拾取模式都使用**透明全屏遮罩**（仅顶部 56px 高的半透明信息条），让用户能清楚看到屏幕内容便于精确定位：

- **拾取坐标**（点击点用）：**单击**屏幕拾取一个点
- **框选区域**（监控区域用）：**按住鼠标左键拖拽**画矩形，松开确认；拖拽过程中显示绿色矩形 + 实时尺寸标签（如 `320 x 80`），自动填入 top/left/width/height
- ESC 任意时刻取消；矩形 < 3x3 视为误操作丢弃

## 停止

点击「停止」按钮：当前一轮跑完后停止监控。
